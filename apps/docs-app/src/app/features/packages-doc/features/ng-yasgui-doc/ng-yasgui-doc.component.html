<cz-yasgui> is an angular wrapper around yasgui . Here is a basic example of it's use without any input:

@if (displayYasgui) { }

Here are some issues you might want to take into consideration when using this component/module:

  • Beware that this component is quite heavy, so it's is recommended to import NgYasguiModule only in lazy loaded modules to minimize the impact on the UX.
  • At the moment, only one instance of <cz-yasgui> can be used on a page, the second instance doesn't seem to render properly, might be a limitation of yasgui itself

API reference for cz-yasgui

{{ "import { NgYasguiModule } from '@cognizone/ng-yasgui';" }}
