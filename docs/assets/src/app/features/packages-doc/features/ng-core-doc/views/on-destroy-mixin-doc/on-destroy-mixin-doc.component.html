OnDestroyMixin() provides an easy to use observable that is fired at component/directive/pipe destruction. The most direct use case for this is to use it in a takeUntil() close like below.

Since it is the most common use case, OnDestroyMixin() comes with a helper method called untilDestroyed() which is just a wrapper for takeUntil(this.onDestroy$)

There are 2 more helpers that can be quite useful:

  • The firstUntilDestroyed() method which provides a combination of first() and untilDestroyed()
  • The OnDestroy$ class that is just a quick wrapper for OnDestroyMixin(class{{ '{' }}})
